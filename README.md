# 🤖 AIスクール

**AIを基礎の基礎から、使える力へ。**

AI初心者が、AIの仕組み・用語・使い方・実践方法を、順番に学べるWebアプリです。

---

## 特徴

- 📚 **12本のレッスン** — 基礎から実践・倫理まで体系的に学べる
- ⚡ **6つの実践ミッション** — コピペ用プロンプト付きのハンズオン体験
- 🛠️ **10のAIツール紹介** — ChatGPT・Claude・Geminiなど主要ツールを解説
- 🏆 **XP＆バッジシステム** — レッスン完了で+50XP、ミッション完了で+100XP
- 💾 **LocalStorage保存** — ログイン不要、データは端末に自動保存
- 📱 **PWA対応** — スマホのホーム画面に追加してアプリとして使える

## 技術スタック

- **フロントエンド**: React 19 + Vite 8
- **スタイル**: Vanilla CSS（CSS変数 + クラス）
- **状態管理**: React useState + LocalStorage
- **外部依存**: なし（APIキー不要）

## ローカルで動かす

```bash
npm install
npm run dev
```

→ http://localhost:5173 で開く

## ビルド

```bash
npm run build
npm run preview
```
